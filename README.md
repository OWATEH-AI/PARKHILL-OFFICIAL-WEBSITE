# Parkhill College - Demo Website

This is a premium, modern, and high-trust demo website built for Parkhill College.

## Features
- **Premium Design**: Navy Blue and Gold layout styled for academic excellence.
- **Interactive Chatbot**: AI-powered assistant to answer parents' queries about fees, registration, and location.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop.
- **WhatsApp Integration**: Floating button for instant parent-school communication.
- **Local Context**: Content and imagery curated for the Zimbabwean education market.
- **Dual Curriculum**: Offers both ZIMSEC and Cambridge curriculum.
- **FAQ Section**: Comprehensive answers to common parent questions.
- **Contact Form**: Direct communication with WhatsApp integration.
- **Gallery**: Showcase of school facilities and activities.
- **Newsletter**: Stay updated with school news and events.

## How to View
1. Open `index.html` in any web browser.
2. Scroll to see the reveal animations.
3. Interact with the "Parkhill Assistant" chatbot icon on the bottom right.
4. Click the WhatsApp button to see the auto-filled enrollment inquiry.

## Technical Details
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), and Javascript.
- **Animations**: Custom scroll-reveal and micro-interactions.
- **SEO**: Meta tags and semantic structure optimized for school searches.
